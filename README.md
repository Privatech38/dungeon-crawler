# Dungeong Crawler
This is a dungeon crawler game written in TypeScript using WebGPU for rendering.

## Running
To run this project you Node.js 24. Firstly run
```shell
npm install
```
To install the necessary dependencies. To run the project, simply run
```shell
npm run dev
```

## ECMAScript/JavaScript
If you need the code in ECMAScript, run:
```shell
npx tsc
```
